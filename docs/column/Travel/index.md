### 经验总结
1. **法律意识**：遇到类似公司要增强法律意识，采取录音、截图等方式留存证据。
2. **面试准备**：提前了解公司情况，面试时问清楚所有问题。
3. **工作选择**：不接受夜班，避免加入不正规的公司。

### 面试题总结
1. **html**：http 协议、web 攻击方式、DNS、http 和 https 的区别、状态码等。
2. **css**：定位方式、垂直居中方法。
3. **js**：对象合并、数组操作、基本数据类型和引用数据类型的区别、递归等。
4. **es6**：es6 新特性、Promise、同源与不同源、跨域方式等。
5. **Vue**：路由模式、同源、跨域、watch 和 computed 的区别、vuex 状态、mutations 的用法等。
6. **大数据可视化，echarts**：常见属性、适配方法。
7. **小程序**：第三方 ui 框架、生命周期、打包问题、上线流程等。
8. **后管（中台）系统**：token 存放位置、动态路由用途、权限判断、前置导航守卫等。
9. **其他**：ts、jQuery 事件委托、D3.js、three.js、Antdv 框架、uview、前端性能优化等。
